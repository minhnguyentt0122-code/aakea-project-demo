### F-02.3 Product Detail Page

**Summary**

The primary product page with all purchase-decision information: high-fidelity image gallery, material/finish configurator, dimension table (inches + cm), lead time for selected configuration, specification PDF, delivery checker, and customer reviews.

**Background / Context**

The product detail page is the most important conversion surface in the HomeStyle storefront. Premium design buyers need complete visual and technical information before committing to a high-value purchase. Missing information — material close-ups, accurate lead times, white-glove delivery confirmation — causes abandonment. The configurator is the centrepiece: every option change must update price, lead time, and availability in real time.

**User Story**

As a customer evaluating a premium design piece, I want to see all details, material options, dimensions, lead times, and specification documents in one place so that I can make a confident purchase decision.

**Flow — Happy Path**

1. Customer navigates to `/products/[slug]` (SSR-rendered for SEO).
2. Page shows: high-fidelity image gallery with material close-up shots, product configurator (material, finish, fabric grade, base type, size), price in local currency, lead time for selected configuration, dimension table (inches + cm), designer credit and bio excerpt.
3. Customer selects configuration options — price, lead time, and stock status update dynamically.
4. Customer clicks 'Download Specification PDF' → server generates TC-09: 15-min S3 presigned URL. Full-resolution spec PDF for verified trade accounts; standard spec for consumers.
5. Customer enters ZIP/postal code in delivery checker → sees estimated delivery dates, methods, and white-glove availability.
6. Customer clicks 'Add to Cart' → item added with selected configuration.
7. Customer scrolls to reviews section showing aggregated rating and approved reviews.

**Flow — Edge Cases**

| Edge Case | Expected System Behaviour |
|---|---|
| Selected configuration is out of stock | 'Add to Cart' disabled. 'Notify Me' button shown (or 'Made to Order' if OD-03 enabled). |
| Product is archived (`is_archived=true`) | Page returns HTTP 301 redirect to parent collection page. |
| TC-09: Spec PDF presigned URL expired before click | URL returns S3 403. Error message: 'Download link expired. Please refresh the page.' |
| Customer views product with no configurations | No configurator shown. Single price and stock status displayed directly. |

**Business Rules**

| BR # | Rule |
|---|---|
| BR-01 | TC-09: Specification PDF served via S3 presigned URL with 15-minute TTL. Full-resolution version for verified trade accounts; standard version for consumers. |
| BR-02 | Configuration selection updates URL query params (e.g. `?material=walnut&finish=natural-oil&size=large`) for shareability. |
| BR-03 | Lead time displayed per selected configuration — not per product. A sofa in standard fabric may show '6–8 weeks'; in special-order leather '12–16 weeks'. |
| BR-04 | Product page SSR renders all SEO fields: meta title, meta description, og:image, JSON-LD Product schema with price, availability, rating, and designer. |
| BR-05 | Designer credit includes name and a brief bio excerpt linking to their designer profile collection page. |
| BR-06 | Reviews shown: approved reviews only, sorted by most helpful (upvotes) by default. |

**Input Validation Rules**

| Field | Type | Required | Validation Rules | Error Message Shown |
|---|---|---|---|---|
| slug (URL param) | String | Yes | Must match a product in DB. Non-existent → 404. Archived → 301. | 404 or 301 |
| ZIP/postal code (delivery checker) | String | No | US ZIP: 5 digits. UK postcode: standard format. EU: 4-6 digits. | 'Enter a valid ZIP / postal code' |
| configurationId (cart add) | UUID | Yes (to add to cart) | Must be a valid configuration UUID belonging to this product. | 'Please select all product options' |

**UI / Display Rules**

| Condition / State | UI Component | Display Text | Colour / Style | Available Action |
|---|---|---|---|---|
| In stock | Stock badge | 'In Stock' | Green badge (#1D9E75) | 'Add to Cart' enabled |
| Low stock | Stock badge | 'Only [N] left in this configuration' | Amber badge (#EF9F27) | 'Add to Cart' enabled |
| Out of stock (no made-to-order) | Stock badge + CTA | 'Out of Stock' | Red badge (#E24B4A) | 'Notify Me When Available' |
| Made to order (OD-03) | Badge + CTA | 'Made to Order — Lead time: [X] weeks' | Blue badge (#185FA5) | 'Add to Cart' with lead time lock |
| No configuration selected | 'Add to Cart' button | 'Select all options above' | Grey disabled button | Disabled until all configurator options selected |
| TC-09: Spec PDF expired | Error message | 'Download link expired. Refresh the page to try again.' | Amber inline message | Page refresh |
| Trade price (trade account logged in) | Price display | 'Trade Price: [price]' below RRP | Subtle secondary line | — |

**API Contract**

| Method | Endpoint | Description | Request Body / Params | Response |
|---|---|---|---|---|
| **GET** | /api/v1/products/:slug | Full product detail | — | `200: { product, configurations[], reviews[], related[] }` |
| **GET** | /api/v1/products/:id/spec-pdf | Generate presigned PDF URL | `?configurationId=` | `200: { url, expiresAt }` (TC-09: 15 min) |
| **POST** | /api/v1/products/:id/notify-me | Back-in-stock subscribe | `{ email, configurationId }` | `200: { message }` |
