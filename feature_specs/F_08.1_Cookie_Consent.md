### F-08.1 Cookie Consent

**Summary**

GDPR-compliant cookie consent banner captures and records user consent before any non-essential cookies are set. Required before EU launch.

**Business Rules**

| BR # | Rule |
|---|---|
| BR-01 | TC-36: No non-essential cookies set before consent recorded. Consent stored with timestamp and policy version. |
| BR-02 | Granular consent: necessary (always on), analytics, marketing. |
| BR-03 | Consent stored in first-party cookie and, if authenticated, synced to user record. |
| BR-04 | TC-37: CCPA 'Do Not Sell My Personal Information' link shown to users in California. Opt-out takes effect immediately (same page load). |

**Permission Matrix**

| Method | Endpoint | Description | Guest | Customer | Trade | Ops | Content | Super Admin |
|---|---|---|---|---|---|---|---|---|
| **POST** | /api/v1/consent | Record consent | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| **GET** | /api/v1/consent | Get current consent state | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |
| **PATCH** | /api/v1/consent | Update preferences | ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |

**API Contract**

| Method | Endpoint | Description | Request Body / Params | Response |
|---|---|---|---|---|
| **POST** | /api/v1/consent | Record consent | `{ necessary: true, analytics: bool, marketing: bool, version: string }` | `200: { consentId, recordedAt }` |
| **GET** | /api/v1/consent | Get consent | — | `200: { necessary, analytics, marketing, recordedAt, version }` |

**TC References:** TC-36 (consent required before non-essential cookies; 12-month validity), TC-37 (CCPA opt-out immediate)
